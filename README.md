## Development

Below are the steps to contribute. If you have any trouble or questions, please don't hesitate to open an issue.

### Installation

Fork the [repo](https://github.com/MartinBraquet/rct-autism) on GitHub (button in top right). Then, clone your repo
and navigating into it:

```bash
git clone https://github.com/<your-username>/rct-autism.git
cd rct-autism
```


##### Resources


### Submission

Add the original repo as upstream for syncing:

```bash
git remote add upstream https://github.com/MartinBraquet/rct-autism.git
```

Create a new branch for your changes:

```bash
git checkout -b <branch-name>
```

Make changes, then stage and commit:

```bash
git add .
git commit -m "Describe your changes"
```

Push branch to your fork:

```bash
git push origin <branch-name>
```

Finally, open a Pull Request on GitHub from your `fork/<branch-name>` → `MartinBraquet/rct-autism` main branch.
