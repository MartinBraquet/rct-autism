# Personalized Pre-Session Prep in Autism: Technical Paper

This folder contains the technical paper for the **Maya Care and Grow** randomized crossover trial.

## Prerequisites

To run these scripts, you need a LaTeX compiler (or upload it to Overleaf).

## Setup Instructions

Simply compile the LaTeX document with pdflatex or any other LaTeX compiler.

## Structure

- `rho-class`: The template for the manuscript.
- `paper.tex`: The main manuscript file.
- `registration_appendix.tex`: The appendix containing the pre-registration plots.
- `rho.bib`: BibTeX file containing the bibliography.
